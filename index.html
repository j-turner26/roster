Add to your calendar

This downloads a standard calendar file (.ics) containing every shift above, in Sydney time.

iPhone / Apple CalendarOpen the downloaded file from Safari's downloads or the Files app, review the events, then tap Add All. Pick which calendar to add them to.
Android / Google CalendarOpen the file with the Google Calendar app if offered. Otherwise go to calendar.google.com > Settings > Import & export, choose the file and import.
Shift times come from the roster's own Shift Timings legend when present.
If a code can't be matched to a time, it's added as an all-day entry so nothing is missed.
